# AI Financial Intelligence Platform – Zomato

## Project Overview
This project is an AI-powered financial analysis platform built using Python, Excel, Tableau, and Gemini AI. It performs revenue analysis, profit analysis, DCF valuation, growth calculations, and AI-based investment recommendations for Zomato.

## Features
- Revenue Growth Analysis
- Quarterly and Annual Financial Analysis
- Profitability Analysis
- Data Visualization using Matplotlib
- Discounted Cash Flow (DCF) Valuation
- AI-Generated Investment Recommendations
- Financial Insights using Gemini AI

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Excel
- Tableau
- Google Gemini API
- Google Colab

## Business Impact
The project helps investors and analysts understand company performance, estimate intrinsic value, and generate data-driven investment decisions using AI.

## Project Workflow
1. Import financial data from Excel
2. Perform data cleaning and analysis
3. Calculate revenue growth trends
4. Visualize financial performance
5. Execute DCF valuation
6. Generate AI-powered investment insights
